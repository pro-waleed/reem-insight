# Reem Insight

نسخة استعراضية حية لمنصة عربية لدراسة وتحليل السوق اليمنية وتقييم الفرص الاستثمارية، مع تركيز أولي على القطاعات التقنية.

## الصفحات الحالية

- `index.html` الصفحة الرئيسية
- `sectors.html` عرض القطاعات ذات الأولوية
- `opportunities.html` بطاقات الفرص الاستثمارية
- `reports.html` التقارير والباقات التجارية
- `about.html` المنهجية وخارطة التنفيذ

## بنية المشروع

- `styles.css` التنسيق العام
- `data.js` البيانات الأولية للقطاعات والفرص والتقارير
- `script.js` ربط البيانات بالواجهة والتفاعل البسيط

## طريقة التشغيل محلياً

بما أن المشروع ثابت `static`, يمكن فتح `index.html` مباشرة في المتصفح.

## النشر على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub.
2. ارفع ملفات المشروع إلى جذر المستودع.
3. فعّل `GitHub Pages` من إعدادات المستودع.
4. اختر النشر من الفرع الرئيسي ومن جذر المشروع.

## التطوير التالي المقترح

- تحويل `data.js` إلى JSON أو قاعدة بيانات فعلية
- إضافة صفحات داخلية لكل قطاع ولكل فرصة
- ربط نماذج التواصل بخدمة بريد أو CRM
- بناء لوحة إدارة لتحديث المحتوى
